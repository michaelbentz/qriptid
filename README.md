# QRiptid

QRiptid is a simple Android app showcasing recommended architecture, integrating APIs for QR code
creation.

### Demonstrate

- MVVM (Model-View-ViewModel)
- Domain Use Cases
- Room
- Retrofit
- Compose
- Flow API
- Dagger Hilt

### Consumed APIs

- https://goqr.me/api/doc/create-qr-code/
- https://goqr.me/api/doc/read-qr-code/
